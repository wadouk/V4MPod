# Correlate

## Purpose
Match a gpx/nmea file with a log file and pictures

## Image directory
 - root
    - folder_name1/
    - folder_name2/
    - ... folder_names/
    - trace.nmea
    - cam.log
    

## Usage
 - set correlate_with_log.py in PATH
 - In the directory where images and log are run `correlate_with_log.py v4mbike -t 7200`
